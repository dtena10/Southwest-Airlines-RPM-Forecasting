# Southwest Airlines RPM Forecasting
Time series forecasting analysis of Southwest Airlines Revenue Passenger Miles (RPM)

# Project Overview
The goal of this project is to evaluate historiacl RPM patterns and develop a forecast for future passenger demand using time series methods. 

# Objectives
* Examine long-run trend and seasonal patterns in the airline passenger demand
* Test the series for stationarity and apply appropriate transformations
* Compare candidate forecasting models
* Evaluate diagnostic results to support model selection
* Produce a multi-period RPM model

# Methods
* Time Series Visualization
* Log-Difference Transformation
* (P)ACF Analysis
* SARIMA Model Construction
* Forecast Generation & Interpretation
* Diebold-Mariano Comparison Testing

# Tools
* RStudio

# Notes
This project was developed as an academic forecasting exercise.
